## About the project

This website was built using Angular. It features 'About', 'CI/CD', and 'Employee CRUD' sections. For the backend API, ExpressJS was utilized.

**Live Demo**: <a href='https://angular.ferozfaiz.com/'>angular.ferozfaiz.com</a>

### Tech Stack:

- Angular
- Sass / Scss
- Bootstrap
- PostgreSQL
- Kubernetes
- Docker
- AWS ECR
- ExpressJS for API
